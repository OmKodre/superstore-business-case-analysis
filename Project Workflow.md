## Project Workflow

The analysis pipeline follows these steps:

1. **Data Source**
   - Kaggle Superstore Dataset (9,994 transactions)

2. **Data Preprocessing**
   - Python used for cleaning and formatting

3. **Database Storage**
   - Data loaded into MySQL

4. **SQL Analysis**
   - Business analysis using SQL queries

5. **Insights Generation**
   - Margin leakage analysis
   - Product profitability analysis
   - Customer RFM segmentation

6. **Dashboard Visualization**
   - Power BI dashboard for executive reporting
